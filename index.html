How It Works

A simple process with powerful results.

1

Select Text

Find content you want to analyze - news articles, social media posts, or any text.

2

Use Shortcut

Copy the text (Ctrl+C) then press the global shortcut (Ctrl+Alt+Shift+T) to analyze.

3

Review Analysis

Get instant insights on potential manipulation, credibility issues, and reasoning flaws.

4

Make Informed Decisions

Use the detailed recommendations to evaluate the content more critically.
